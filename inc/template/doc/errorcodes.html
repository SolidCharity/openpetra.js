Error Code Structure

An OpenPetra Error Code string consists of four sections:
  • the abbreviated name of the OpenPetra Module in which the error occurs, or 'GEN' if it is not related to any OpenPetra Module, or 'GENC' if it is not specific to the OpenPetra application (Ict.Common* Libraries only);
  • a full stop ('.');
  • a running five-digit number with leading zeroes which is unique;
  • a single character:
    • 'V' denotes a data validation error;
    • 'N' denotes a non-critical error;
    • 'E' denotes any other error.
